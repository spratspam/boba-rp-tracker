Hi!

This is a small desktop app for tracking RP threads, replies, and notes.
It’s designed to be simple, cozy, and easy to use. 

You can use either the installer OR the standalone .exe — both work the same.

✨ What’s New (v1.0.1)
Replies now automatically update:
Last Turn
Status (Waiting on Me / Them)
Archived RPs are hidden from the main view
Added On Hold status option
General fixes and improvements
🧋 How to Use (Portable/No installer)
Double click Boba RP Tracker.exe to launch
Create RPs and add replies
Everything saves automatically

💾 Install (Recommended)
Go to the Releases section

Download:

Boba_RP_Tracker_v1.0.1_Setup.exe
Double-click the file
Follow the install steps
Launch from:
Start Menu, or
Desktop shortcut (if selected)

⚠️ Important
Keep the .exe in the same folder
Do not move or delete files the app creates (like data files)
Your RPs are saved locally on your computer
🛡️ Windows Warning

If Windows gives you a warning:

Click "More info"
Then click "Run anyway"

(This happens because the app isn’t digitally signed — it’s safe!)

💬 Notes
Updates to the app will not delete your RPs
If something looks wrong after updating, try restarting the app

That’s it — enjoy 💚
